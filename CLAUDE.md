# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 基本ルール

### コミュニケーション
- **日本語で全ての説明とやり取りを実施すること**

### 作業フロー
- **作業開始時に必ずCLAUDE.mdファイルを確認すること**
- **作業内容がCLAUDE.mdの更新を必要とする場合は、作業計画にCLAUDE.md更新を含めること**
- **ソースコードを変更した場合は必ずビルドして確認すること**
- **gitのコミットは適切な単位（行単位、ファイル単位、作業単位など）を毎回ユーザに提案し、確認を取ったうえで実施すること**

#### CLAUDE.mdを更新すべきケース
以下の場合はCLAUDE.mdの更新が必要:
- 新しいビルド/テストコマンドの追加
- アーキテクチャの大きな変更
- 新しい重要な設計パターンの導入
- プロジェクト構造の変更
- 開発フローやルールの変更
